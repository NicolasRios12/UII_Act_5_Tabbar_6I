LENGUAJE: DART | FRAMEWORK: FLUTTER
## 1. DESCRIPCION DEL NEGOCIO
Dog Club es una aplicacion diseñada para la gestion de servicios caninos.

## 2. ESPECIFICACIONES DE DISEÑO
La aplicacion sigue una paleta de colores estricta para mantener la identidad visual:

Fondo: Color(0xFFF0F4F8)

Botones: Color(0xFF0D47A1)

AppBar / Iconos: Color(0xFF1976D2)

Acentos / Footer: Color(0xFFBBDEFB)

## 3. ESTRUCTURA DE NAVEGACION
Se implementa una barra de navegacion inferior (TabBar/BottomNavigationBar) con 5 opciones principales:

Guarderia: Gestion de juegos y cuidado diario.

Hospedaje: Control de estancia nocturna.

Paseos: Seguimiento de rutas y ejercicio.

Perfil: Datos del propietario.

Mascota: Informacion detallada del canino.

## 4. CARACTERISTICAS TECNICAS
AppBar: Titulo centrado con un "Icon.pets" en la propiedad leading.

Cuerpo: Imagen central cargada desde red (Image.network) para cada seccion.

Navegacion: Manejo de estados mediante un indice seleccionado en el archivo principal.
